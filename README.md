## Tech

Deliberately dependency-free: a single self-contained `index.html` with all styles and scripts inline — no framework, no build step, no bundler.

- Vanilla **HTML, CSS, JavaScript**
- Typography: Sora, Source Serif 4, IBM Plex Mono (Google Fonts)
- Light/dark theming via CSS custom properties + `localStorage`
- Progressive enhancement: scroll-reveal animations, lightbox galleries, animated counters, typewriter, contact form (Formspree)
- Hosted on **GitHub Pages**

## Run locally

No tooling required — open the file directly:

```bash
git clone https://github.com/MarkhasRisen/mark-profile.git
cd mark-profile
# open index.html in your browser
```
